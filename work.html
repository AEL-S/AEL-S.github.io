Projects

PacMen

Javascript, CSS, and HTML are combined to create an interactive page where the user can add as many PacMen as they would like, and make them all move independently when the Start Game button is clicked.

Everything required to run this project is in the repository. You're welcome to fork and clone: PacMen Repository or you can just click the image above to navigate to the project site itself. Press Add Pacman, then press Start Game to see them move!

EyeExercise

Here, Javascript, CSS, and HTML are combined to create an interactive page where the eyes follow the mouse around the page.

Feel free to fork and clone the respository at: EyeExercise Repository or click the image above to open the full-screen project page. Move your cursor and see the eyes watch your every move!

RealTimeBusTracker

Like those above, this assigment was also completed for the MIT xPro Fullstack Development with MERN certificate program. Buses on a route in Boston are tracked in real time by calling to an API service.

To run this project, fork and clone this repository and enter your API key on the mapanimation.js file.

**Since this project requires your own personal API key, the image here will take you to the repository instead of an active GitHub pages page like the previous two projects.**
